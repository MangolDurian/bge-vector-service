# 使用文档

> 中文文本向量服务的安装、启动与调用指南

---

## 1. 环境要求

| 依赖 | 最低版本 | 说明 |
|------|---------|------|
| Python | 3.9+ | 推荐 3.10 或 3.11 |
| pip | 最新版 | 包管理器 |
| （可选）CUDA | 11.7+ | GPU 推理加速 |

---

## 2. 安装步骤

```bash
# 进入项目目录
cd /Users/mangguo/code/2026/May/embedding-api

# 建议使用虚拟环境
python -m venv venv
source venv/bin/activate

# 安装依赖
pip install -r requirements.txt
```

首次调用时模型会自动从 HuggingFace 下载（约 400MB），后续使用本地缓存。

---

## 3. 启动服务

### 方式一：直接启动

```bash
python main.py
```

### 方式二：uvicorn 启动（推荐生产环境）

```bash
# 单进程
uvicorn main:app --host 0.0.0.0 --port 7999

# 多 worker（需要确保机器内存/显存充足）
uvicorn main:app --host 0.0.0.0 --port 7999 --workers 2
```

### 方式三：通过环境变量自定义配置

```bash
# 切换为 GPU 推理
export EMBEDDING_DEVICE=cuda

# 修改端口
export EMBEDDING_PORT=9000

# 修改 batch size
export EMBEDDING_BATCH_SIZE=64

# 启动
python main.py
```

所有可配置项见 `config.py`，均可通过环境变量覆盖。

## 4. 调用示例

### curl

```bash
curl -X POST http://localhost:7999/embed \
  -H "Content-Type: application/json" \
  -d '{"texts": ["自然语言处理", "机器学习"]}'
```

### Python（requests）

```python
import requests

resp = requests.post(
    "http://localhost:7999/embed",
    json={"texts": ["自然语言处理", "深度学习"]},
)
data = resp.json()
print(f"向量维度: {data['dim']}")      # 768
print(f"向量数量: {data['count']}")     # 2
print(f"第一条向量前5位: {data['embeddings'][0][:5]}")
```

### Python（计算相似度）

```python
import requests
import numpy as np

resp = requests.post(
    "http://localhost:7999/embed",
    json={"texts": ["今天天气真好", "今天阳光明媚"]},
)
embs = np.array(resp.json()["embeddings"])

# 内积相似度
similarity = np.dot(embs[0], embs[1])
print(f"相似度: {similarity:.4f}")
```

### JavaScript / Node.js

```javascript
const resp = await fetch("http://localhost:7999/embed", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ texts: ["你好世界", "测试文本"] }),
});
const data = await resp.json();
console.log(`维度: ${data.dim}, 数量: ${data.count}`);
```

### 前端 UI（可选）

`web/index.html` 是辅助测试页面，运维部署时可部署也可不部署。若需要通过端口直接访问前端，可启动静态文件服务：

```bash
python -m http.server 8001 -d web
```

浏览器访问 `http://localhost:8001` 即可进入前端页面。前端页面只负责辅助调用接口，后端 API 仍使用 `7999` 端口。

## 5. 常见问题

### Q: 下载模型很慢怎么办？

设置 HuggingFace 镜像：

```bash
export HF_ENDPOINT=https://hf-mirror.com
python main.py
```

### Q: 没有 GPU 能用吗？

可以。默认会自动使用 CPU 推理，速度稍慢但功能完全一致。

### Q: 如何调整单次最大文本数量？

修改环境变量 `EMBEDDING_MAX_TEXTS`，默认 128 条。

### Q: 启动后内存/显存占用多少？

bge-base-zh-v1.5 模型约 400MB，CPU 推理约占用 1~2GB 内存，GPU 推理约占用 500MB 显存。
